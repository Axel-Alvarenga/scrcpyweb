# 📱 SCRCPY Web

Aplicación web para controlar tu teléfono Android con scrcpy desde el navegador.

## ✨ Características

- 🔗 Conectar teléfono por Wi-Fi
- 📱 Ver pantalla del teléfono en PC
- 🎬 Configurar calidad de video (Baja/Media/Alta/Ultra)
- 🪟 Configurar ventana (Tamaño, Pantalla completa, Siempre encima)
- 📲 Instalar APK desde PC
- 📊 Ver información del teléfono
- 🔌 Activar Modo WiFi desde la app (sin CMD)
- 🛠️ Gestión completa de opciones

## 🚀 Descarga

Descarga el ejecutable desde [Releases](https://github.com/TU_USUARIO/scrcpy-web/releases)

## 📋 Requisitos

- Windows 7/10/11
- Teléfono Android con depuración USB activada
- Conexión WiFi (misma red que la PC)

## 📱 Configuración inicial (solo una vez)

1. Conecta tu teléfono por USB a la PC
2. Abre SCRCPY-Web.exe
3. Haz clic en "Activar Modo WiFi"
4. ¡Listo! Ya puedes desconectar el USB

## 🎮 Uso diario

1. Abre SCRCPY-Web.exe
2. Pon la IP de tu teléfono
3. Haz clic en "Conectar y abrir scrcpy"

## ⚠️ Solución de problemas

| Problema | Solución |
|----------|----------|
| No conecta | Verifica que PC y teléfono estén en la misma red WiFi |
| Error ADB | Ejecuta "Activar Modo WiFi" con el teléfono conectado por USB |
